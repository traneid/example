Программа для заполнения ТС на сайте мос.ру

В данный момент еще не законченная программа.
Программа будет в полу автоматическом состоянии заполнять данные по транспорту на сайте мос.ру




тело программы 
поля = ('ГРЗ', 'Серия номер СТС', 'Дата выдачи СТС', 'VIN', 'Масса без нагрузки',
                 'Максимальная допустимая масса', 'Год выпуска', 'Номер диагностической карты',
                 'Срок действия | До какого числа действует', 'Марка модель АМ', 'Экологический класс', 'Категория ТС')

Получить данные по тс = через requests стучит в нашу базу (1с Авто-Пэк) (сервис по отдаче информации был написан программистом компании), по вин номеру получает информацию о транспорте и заполняет поля (все поля можно редактировать или дополнить, логист просто проверяет и по надобности корректирует)
Кнопка не работает вне сети ПЭК !!!

Собственник тс = Через radiobutton получает информацию (совпадает ли собственник или нет) если да, поле ниже блокируется и не заполняется, если нет нужно указать в поле организацию на которой находиться транспорт.

Добавить документ = Выходит дочернее окно программы, где нужно выбрать из списка нужный документ (список взят с оф. сайта мос.ру)

Добавить файл = Добавляет в программу файл (точнее путь до нее), без выбора типа документа файл не добавить.

После добавления, мы увидим его в отдельном поле (тип и имя документа)

Удалить = кнопка появляется после добавления документа, если на нее нажать, удалит последний документ, если документов нет, скрывается

Заполнить информацию на сайте = Блок будет отвечать за заполнение информации на сайте, без участия пользователя с помощью selenium, вход по действующей эцп, установка плагина для работы, заполнения данных (в данный момент можно только получить готовый справочник по которому будет заполняться информация)
