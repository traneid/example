Программа калькулятор котенка (подсчет расходов за котиком).

Основные элементы программы.


Главное окно.


Динамическая линия подсчета состоящая из 4 заполняемых полей (Товар, Количество, цена товара, количество дней (на расход)) и итог (вычисляемое поле), а так же одной кнопки равно (=) 
Итог вычисляется по формуле = Количество * Цена_товара / Количество_дней

Поле "Товар" = Динамически проверяет количество символов в поле символов в поле должно быть не меньше 3, если ошибка поле станет красным (есть баг в поле при указании одного символа нет подсветки)

Кнопка равно = по умолчанию заблокирована, пока не будут заполнено корректно поля

Количество дней = установка проверки деления на 0

При заполнении любой ячейки, добавляется новая линия для заполнения с таким же функционалом.

При отчистках полей, линия удаляется.


Статические элементы нижней части программы.
Всего затраты за год, кнопка равно суммирует ответы из динамических полей и умножает на 365 (не трогает пустые ответы или с ошибкой)

Количество лет = поле по умолчанию заполняется на 1 год, можно указать произвольный 

Общее количество = Берет данные из поля Всего затраты за год * количество лет 


Общее.

Программа получает интерфейс из json файла 

Файл состоит из справочника "table" и "footer"

у справочника "table" есть "header" - заголовки и "line" - линия они содержат двумерный массив с информацией о справочнике в которым указан виджет и его местоположение

у справочника "footer" содержит двумерный массив с информацией о справочнике в которым указан виджет и его местоположение
